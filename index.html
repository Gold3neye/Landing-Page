Data Analyse

Le programme de deux semaines permet d'être autonome en Data Analyse et sur des outils Data Science. Nous verrons :
  • Les outils de Data Visualisation
  • La Web Analytics avec Python
  • L'A/B testing
  • SQL : gérez des bases de données
  • Le Machine Learning en Python
  • Le cloud computing
Réalisez votre premier projet Data Science : Vous mettrez en application tout ce que vous avez appris pendant votre bootcamp. Vous aurez toutes les clés pour continuer dans cette voie des Data Sciences, manager une équipe Data ou démarrer votre projet IA
En savoir plus

Data Science

Le programme de 12 semaines à temps complet est destiné aux personnes souhaitant devenir Data Scientist.
A la fin du bootcamp, vous maitriserez le langage de programmation Python et vous serez capable de l'utiliser pour extraire des données et les préparer pour pouvoir leur appliquer des modèles de Machine Learning supervisés et non supervisés.
Vous allez acquérrir aussi des connaissances solides dans la gestion de bases de données SQL et NoSQL et vous serez capables de gérer des problématiques Big Data avec les outils cloud et Spark.
Les 12 semaines se conclueront par un projet de Data Science mené de A à Z que vous présenterez pour valider la formation.

Data Engineering

70 heures pour obtenir les compétences d’un Data Engineer, extrêmement demandés :
  • Cloud Computing : maîtrise d'AWS
  • Processus ETL & Data storing
  • Données en streaming
  • Déployer un modèle en production
    • 2 projets de Data Engineering pour parfaire votre portfolio : automatisation de process et mise en production de modèles. Vous serez extrêmement polyvalent sur les différents pans du pipeline Data depuis sa construction : Data Science & Data Engineering.
